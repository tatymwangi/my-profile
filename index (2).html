Welcome to plain-html-blog

By Tatiana

April 1, 2020

plain-html-blog is a very clean Jekyll blog that works without CSS. It was inspired by websites which (for some valid reason) make minimal use of styling with CSS and the like.

Just to get an idea check out https://www.beyonce.com, Beyonce's personal website, www.google.com or https://zendaya.com. They all come with a certain 90s flavour which in 2020 is stil very functional and in a way elegant (only 90s developers will know...)

Other than that, this Jekyll blog works like any other static website so you should feel right at home. So now go ahead, you'll want to get the source code and read the README to learn how to make your own plain-html-blog.
